What is GIS?
Stands for: Geographic Information Systems

A geographic information system (GIS) is a system designed to capture, store, manipulate, analyze, manage, and present all types of geographical data. The key word to this technology is Geography – this means that some portion of the data is spatial. In other words, data that is in some way referenced to locations on the earth.

Find more here: https://researchguides.library.wisc.edu/GIS
What are common workflows for nonprofits mapping?
  • Putting data into a web map to show to the public
  • Collecting data in the field
  • Give mobile maps to volunteer locations
  • Use internal maps to track internal data
  • Complete analysis about location of work and it's impact
  • Show projects on map to show stakeholder/development progress
What are the best software options?
It really depends on your need.

1) It's worth noting that Esri is essentially the largest provider of GIS software in the world. This makes them a great choice for a large number of use cases. They also run a non-profit program that gives registered 501c3's access to most of the tool that you need to complete most tasks, including: data collection, cloud storage, easy templates to create web maps (no programming required) as well as a license for their desktop software. There are some limitions on the number of users, but overall this is a great choice for many non-profit organizations.

2) If you need low cost, but can't (or don't want to) use Esri's software through their non-profit program, open source is another great option. It won't give you as many tools, and will mostly be desktop based - using QGIS. This can help get you mapping and using GIS fast and for free. Click here to see the QGIS basics.

3) Lastly, sometimes you just need something unique. This would get you into the world of custom GIS configurations and/or web maps and apps. This can be a bit more costly if you need to hire someone, but if you have a background in any programming you can do a lot fairly quickly. The best option here is talk to someone who can help decipher the techno-jargon and help you build all the details that you have in mind. You can always contact us for assitance or to chat about ideas.
How much does it cost to start mapping?
This answer can vary depending upon need. The bottom line: it can be free to start mapping.

Options:

Use open source GIS tools:
  • Cost: free
  • QGIS, Leaflet, Open Street Map, GeoDa
Use private GIS tools/suites:
  • Cost: possible free or usually reduced
  • Esri Non-profit Program - ArcGIS, CARTO, MapBox, GoogleMaps, Hexagon Geospatial, and many more!
Custom apps:
  • Cost: can get pricey (unless you have a developer friend or volunteer)
  • These can be simple, one-off apps or very large custom systems

How can I collect data?
There are a few ways to accomplish this:

  • Try the Map501 Collector to edit and export simple data.
  • If using Esri and their non-profit program, you will have access to Collector for ArcGIS and Survey123. This makes it very easy to get started collecting data in the field and syncing it back to your main database.
  • There are a few other paid apps that also do a good job: Fulcrum, Trimble TerraFlex, MapItGIS.
